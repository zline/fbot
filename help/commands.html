Общие команды: !help, !ident, !version, !date, !time, !uptime, !seen, !send, !register

Команды пользователя: !send, !read, !unident

Команды войса канала: !voice, !devoice

Команды полуоператора канала: !hop, !dehop

Команды оператора канала: !names, !topic, !invite, !op, !deop, !mass, !kick, !ban, !unban, !chanaccess

Команды администратора канала: !bot, !seenchan, !hello, !dialogue, !talksetup, !talk, !act, !say,
!chanaccess, !phrasereaction, !useglobalbase, !badwords

Команды Global Operator'a: доступ оператора к любому каналу + !acclist

Команды Root Administrator'a: доступ администратора к любому каналу + !id, !join, !part, !rejoin, !perform,
!ijoin, !ujoin, !talkfreq, !com, !quit, !mact, !amsg, !acclist, !adduser, !deluser, !nick, !rename, !reboot =====================================

!help - выводит справку по командам бота
Команда на канале, в привате: !help [команда]
'команда' - команда, помощь по который Вы хотите получить. Если опущена, выводится список всех команд.
=====================================================================

 !ident - идентифицирует Вас к боту
Команда в привате: !ident пароль
* логин - Ваш ник
=====================================================================

 !version - выводит версию бота
Команда на канале, в привате: !version
=====================================================================

 !date - выводит текущую дату
Команда на канале, в привате: !date
=====================================================================

 !time - выводит текущее время
Команда на канале, в привате: !time
=====================================================================

 !uptime - выводит информацию о времени работы системы и времени, проведенном ботом в сети
Команда на канале, в привате: !uptime
=====================================================================

 !seen - поиск пользователя в сети
Команда действует на канале и в привате.
!seen без параметров - вывод информации о размере базы
!seen маска (напр. !seen us*r) - поиск пользователя по маске
!seen пользователь - поиск указанного пользователя
=====================================================================

 !send - отправка сообщения пользователю
Команда действует только в привате.
!send ник сообщение
=====================================================================

 !read - чтение сообщений
Команда действует только в привате.
Формат: !read
*Примечание: вы должны быть зарегистрированы и проидентифицированы (!ident)
=====================================================================

 !id - бот идентифицируется к сервису ников
Команда в привате: !id
=====================================================================

 !join - вход на канал(ы)
Команда на канале, в привате: !join канал1[,канал2,...] [пароль_от_канала]
=====================================================================

 !part - выход с канала(ов)
Команда на канале: !part [канал1,канал2...]
Команда в привате: !part канал1[,канал2...]
* при использовании на канале без параметров - выход с канала
=====================================================================

 !rejoin - перезаход на указанный канал
Команда на канале: !rejoin [канал]
Команда в привате: !rejoin канал
* при использовании на канале без параметров - перезаход на канал
=====================================================================

 !names - вывод списка пользователей указанного канала
Команда в привате: !names канал
* бот должен находится на указанном канале
=====================================================================

 !perform - управление листом автовыполнения
(список irc-команд, посылаемых серверу сразу после подключения)
Действует в привате.
!perform list - получение списка
!perform add команда - добавление команды в список
!perform del № - удаление команды по ее номеру
=====================================================================

 !bot - включение/выключение бота поканально
Команда на канале: !bot off - выключение бота
Команда в привате: !bot канал on/off - включение/выключение бота на заданном канале
=====================================================================

 !ijoin - управление списком приветствий бота
Действует в привате.
!ijoin list - получение списка
!ijoin add приветствие - добавление приветствия в список
!ijoin del № - удаление приветствия по его номеру
=====================================================================

 !ujoin - управление списками персональных приветствий
Действует в привате.
!ujoin без параметров - получение списка персонально приветствуемых пользователей
!ujoin пользователь list - получение списка приветствий заданного пользователя
!ujoin пользователь add приветствие - добавление приветствия для пользователя
!ujoin пользователь del приветствие - удаление приветствия для пользователя
=====================================================================

 !seenchan - включение/выключение фунцкии SEEN поканально
Команда на канале: !seenchan on/off - включение/выключение функции соответственно.
Команда в привате: !seenchan канал on/off - включение/выключение функции на заданном канале
=====================================================================

 !unident - закрытие доступа к боту.
Используется в привате без параметров. Команда нужна лишь в исключительных случаях
(например когда в сети не работает /watch и Вы отключили режим extra_secure)
(более подробно о системе безопасности бота - см. документацию)
Для последующего открытия доступа используйте !ident
=====================================================================

 !talk - включение/отключение разговора бота на канале
Бот отвечает в среднем на каждую n-ю фразу (n - устанавливается командой !takfreq).
Команда на канале: !talk on/off
Команда в привате: !talk канал on/off
=====================================================================

 !hello - включение/отключение режима приветствий и прощания с пользователями
Бот приветствует вновьвошедших нотисом, и отвечает на прощания
Команда на канале: !hello on/off
Команда в привате: !hello канал on/off
=====================================================================

 !dialogue - включение/отключение ответов бота на обращение к нему
Бот отвечает, если в чьей-либо фразе присутствует его ник или один из
его альтернативных ников (см talk.conf, секция [dialogue_altnicks])
Команда на канале: !dialogue on/off
Команда в привате: !dialogue канал on/off
=====================================================================

 !talkfreq - устанавливает частоту разговора бота на каналах (см !help !talk)
Команда на канале, в привате: !talkfreq частота
Частота должна быть целым числом больше 0
=====================================================================

 !talksetup - отображает настройки разговорных функций для заданного канала
Команда на канале: !talksetup
Команда в привате: !talksetup канал
=====================================================================

 !act - совершение действия на канале от 3-го лица (/me)
Команда в привате: !act канал сообщение
=====================================================================

 !say - отправить сообщение на указанный канал
Команда в привате: !say канал сообщение
=====================================================================

 !com - отправка команды irc серверу
Команда в привате: !com команда
*Примечание: команда должна быть без префикса / , например: !com JOIN #foo
=====================================================================

 !quit - выход из irc, закрытие программы
Команда в привате, без параметров
=====================================================================

 !mact - отправка сообщения от 3-го лица (/me) на все каналы
Команда в привате: !mact сообщение
=====================================================================

 !amsg - отправка сообщения на все каналы
Команда в привате: !amsg сообщение
=====================================================================

 !topic - смена топика канала
Команда на канале: !topic топик
Команда в привате: !topic канал топик
=====================================================================

 !invite - приглашение пользователя на канал
Команда на канале: !invite ник
Команда в привате: !invite канал ник
=====================================================================

 !op - дать оп пользователю
Команда на канале: !op ник
Команда в привате: !op канал ник
*Примечание: если параметр ник опущен, оп дается Вам
=====================================================================

 !deop - забрать оп у пользователя
Команда на канале: !deop ник
Команда в привате: !deop канал ник
*Примечание: если параметр ник опущен, оп забирается у Вас
=====================================================================

 !hop - дать хоп пользователю
Команда на канале: !hop ник
Команда в привате: !hop канал ник
*Примечание: если параметр ник опущен, хоп дается Вам
=====================================================================

 !dehop - забрать хоп у пользователя
Команда на канале: !dehop ник
Команда в привате: !dehop канал ник
*Примечание: если параметр ник опущен, хоп забирается у Вас
=====================================================================

 !voice - дать войс пользователю
Команда на канале: !voice ник
Команда в привате: !voice канал ник
*Примечание: если параметр ник опущен, войс дается Вам
=====================================================================

 !devoice - забрать войс у пользователя
Команда на канале: !devoice ник
Команда в привате: !devoice канал ник
*Примечание: если параметр ник опущен, войс забирается у Вас
=====================================================================

 !mass - действие, применяемое ко всем(*) пользователям канала
Команда на канале: !mass опция
Параметр опция имеет следующие значения:
op - дать всем оп
deop - забрать у всех оп
hop - дать всем хоп
dehop - забрать у всех хоп
voice - дать всем войс
devoice - забрать у всех войс
kick - кикнуть всех
*Примечание: опции deop, dehop, devoice, kick не действуют на администраторов бота и самого бота.
=====================================================================

 !acclist - просмотр листа доступов к боту
Команда в привате, без параметров
=====================================================================

 !adduser - добавление нового пользователя в лист доступа
Команда в привате: !adduser уровень ник пароль
Возможные уровни: operator, user
=====================================================================

 !deluser - удаление пользователя из листа доступа
Команда в привате: !deluser ник
=====================================================================

 !nick - смена ника бота
Команда на канале и в привате: !nick ник
*Примечание: изменения ника не сохраняются в конфигурации
=====================================================================

 !rename - переименование бота и изменение пароля с сохранением изменений
Команда в привате: !rename ник [пароль]
*Примечание: пароль - пароль на nickserv'e, необязательный параметр, если он не задан, меняется только ник
* изменения сохраняются в конфигурации
=====================================================================

 !kick - кикнуть пользователя с канала
Команда в привате: !kick канал ник [причина]
Команда на канале: !kick ник [причина]
*Примечание: параметр причина необязателен
* не действует на администраторов бота
=====================================================================

 !ban - забанить пользователя на канале
Команда в привате: !ban канал ник/маска
Команда на канале: !ban ник/маска
*Примечание: если указана маска, бот ставит бан по ней, если указан ник - бот ставит бан на ник, и, если есть, на его адрес
* если включена настройка Авто-кика, забаненный пользователь будет кикнут
=====================================================================

 !unban - снять с себя бан / все баны на канале
Команда в привате: !unban канал [all]
* если указан необязательный параметр all , снимаются все баны канала
=====================================================================

 !chanaccess - управление локальными доступами
Команда в привате: !chanaccess канал admin|oper|hop|voice add|del|list [ник]
Опции admin|oper|hop|voice - соответствуют администратору, оператору, полуоператору, войсу канала
add - добавить доступ, del - удалить, list - просмотреть доступы
команда !chanaccess без параметров - просмотр ваших локальных доступов
Примеры: !chanaccess #chan admin add nick - добавление администратора nick на канал #chan
!chanaccess #chan oper list - просмотр списка операторов канала #chan
* изменять доступы администраторов канала могут только Root Administrator'ы, остальные доступы меняются администраторами каналов, просматривать списки доступа могут операторы канала.
Для получения справки о уровнях доступа наберите !help access
=====================================================================

 !phrasereaction - включение/отключение режима реакций на фразы
Бот реагирует на определенные фразы, выдавая определенные ответы
Команда на канале: !phrasereaction on/off
Команда в привате: !phrasereaction канал on/off
=====================================================================

 !useglobalbase - разрешение/запрещение использования глобальных фраз разговорных функций на канале
По умолчанию на канале бот использует фразы из глобальной базы (опция on), что может быть отключено (off). Настройка распространяется на все разговорные функции.
Команда на канале: !useglobalbase on/off
Команда в привате: !useglobalbase канал on/off
=====================================================================

 !badwords - управление функцией антимата
Включение/выключение:
Команда на канале: !badwords on/off
Команда в привате: !badwords канал on/off
Просмотр списка блокируемых слов: !badwords list
Добавление слова: !badwords add блокируемая_фраза
Удаление слова: !badwords del блокируемая_фраза
Изменения действия бота: !badwords action kick|kickban|kick-kickban
kick - кик, kickban - бан и кик, kick-kickban - предупредительный кик, в случае повтора - kickban
=====================================================================

 !reboot - перезагрузка бота
Команда в привате, на канале, без параметров
=====================================================================
